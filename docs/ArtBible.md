# Art Bible

## Visual Goal

A cozy 2D pixel-art guild-town management game with compact, readable characters and buildings.

## Character Direction

- Base character sprite size: 16x16 pixels.
- Style: chibi, readable, simple silhouettes.
- Initial animation goals:
  - Idle
  - Walk down
  - Walk side
  - Walk up

## Modular Character Layers

Future character sprites should support:
- Base body
- Hair
- Hair color
- Shirt/top
- Pants/bottom
- Shoes
- Hat
- Weapon overlay
- Class accessory

## Placeholder Art Rule

During early development, simple colored rectangles or rough Aseprite sprites are acceptable. Do not block gameplay implementation waiting for finished art.

## Pixel-Art Import Settings

When importing pixel art into Godot:
- Disable filtering.
- Use nearest-neighbor scaling.
- Keep sprites aligned to whole pixels where possible.
- Avoid blurry scaling.
