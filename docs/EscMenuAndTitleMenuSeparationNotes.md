# ESC Menu and Title Menu Separation

## In-Game ESC Menu

The ESC menu is for the active game session.

Purpose:

```text
pause
resume
open save/load info
open settings
quit/return placeholder
```

Current ESC buttons:

```text
Resume
Save / Load
Settings
Quit
```

## Future Title/Main Menu

The title/main menu is the landing screen when the game launches.

Future title menu buttons:

```text
Continue
New Game
Load Game
Settings
Quit
```

## Intended Difference

Continue belongs to the title menu and should load autosave.

Resume belongs to the in-game ESC menu and should return to the current session.

## Current Status

Only the ESC in-game overlay exists right now.

The title/main menu is not implemented yet.
