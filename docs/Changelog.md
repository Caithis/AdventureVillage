# Changelog

## v0.4.3 - Placed Building Destination Routing

### Added
- Placed General Store can become active adventurer destination.
- Placed Inn can become active rest/sleep destination.
- Dynamic route marker refresh after building placement.
- Dynamic route marker refresh after building movement.
- Dynamic route marker refresh after building demolition.
- Active Store route visual.
- Active Inn route visual.
- Route labels showing PLACED or FALLBACK.
- Build Mode panel collapse/expand control.
- Placed building routing documentation.
- Build menu UX documentation.

### Changed
- Adventurer spawn routines now refresh dynamic route markers before assigning routes.
- Returned adventurer routines now refresh dynamic route markers before assigning store/inn routes.
- Fixed fallback buildings now serve as fallback route destinations only when no placed version exists.

### Not Yet Added
- Player-selected active building choice.
- Nearest-building routing.
- Building capacity.
- Building stock-aware routing.
- Save/load for placed buildings.
- Scrollable build menu.
