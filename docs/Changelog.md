# Changelog

## v0.4.4 - Building Save / Load Foundation

### Added
- Placed building save file at `user://placed_buildings.json`.
- Auto-save after building placement.
- Auto-save after building movement.
- Auto-save after building demolition.
- Auto-load placed buildings when Town scene starts.
- Manual Save Buildings button.
- Manual Load Buildings button.
- Preserves building type, position, size, and original cost.
- Rebuilds active route markers after loading.
- Building save/load documentation.

### Changed
- `_place_building()` now returns the created building node.
- Build panel is slightly taller to fit save/load buttons.

### Not Yet Added
- Full game save/load.
- Money save/load.
- Adventurer save/load.
- World map save/load.
- Building upgrades/workers save/load.

## v0.4.3 - Placed Building Destination Routing

### Added
- Placed General Store / Inn routing.
- Dynamic route markers.
- Collapsible Build Mode panel.
