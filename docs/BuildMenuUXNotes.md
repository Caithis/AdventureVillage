# Build Menu UX Notes

## v0.4.3 Implementation

The Build Mode panel is now collapsible.

Current controls:

```text
Hide Build Menu
Show Build Menu
```

## Why This Matters

The construction UI will get larger over time.

Future build tools may include:
- building categories
- roads
- decorations
- plots
- upgrades
- demolition mode
- move mode
- material costs
- worker assignments

A non-collapsible panel will block too much of the screen.

## Future Direction

The Build Mode panel should eventually support:
- collapsible sections
- scrollable list
- category tabs
- compact mode
- hotkey toggle
- optional transparency
