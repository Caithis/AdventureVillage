# Future Building Interaction Notes

## Building Hover

Current:
- General Store hover changes highlight visibility.

Future:
- Outline shader or cleaner pixel-art border.
- Tooltip with building name.

## Building Click Menu

Current:
- General Store opens a prototype Control menu.

Future:
- All buildings should use a shared menu framework.
- Menu content should be generated from building data.
- Building menus should support tabs:
  - Overview
  - Economy
  - Workers
  - Stock
  - Upgrades
  - Move/Relocate

## Placement and Movement

Long-term:
- Player places Guild Hall at start.
- Tutorial guides Inn and General Store placement.
- Buildings can be moved.
- Roads/decorations can be placed.
- Town land expands through purchasable plots.
