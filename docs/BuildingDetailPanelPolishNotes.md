# Building Detail Panel Polish

## v0.5.2 Implementation

The building menu has been reorganized into readable sections.

## Sections

```text
IDENTITY
CAPACITY / QUEUE
SERVICE
WORKERS
UPGRADES
POLICY
```

## Why This Matters

The building menu now includes:
- material buying policy
- capacity
- queue
- service speed
- worker placeholder controls
- upgrade controls

Without organization, each new system makes the menu harder to use.

## Fixed vs Placed Labeling

The Identity section now distinguishes:
- placed buildings
- fixed fallback buildings
- protected fallback/test buildings

## Current Limitations

Still missing:
- final styling
- icons
- tabs
- scroll support
- condensed labels
- selected-building highlight integration
- dedicated building stats scene/resource
