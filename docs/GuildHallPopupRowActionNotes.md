# Guild Hall Popup Row Action Notes

## v0.6.15

The Guild Hall popup now uses per-adventurer placeholder row buttons.

## Row Actions

Each known adventurer row can show:

```text
Favorite
Request Return
Contract
```

## Current Limitations

The row action UI is still basic.

It proves the control structure but is not final presentation.

## Priority Return Rule

Only one adventurer can be marked for priority return at a time.

This is intentional.

Request Return should eventually require:
- resources
- messenger/notice system
- Guild Hall level
- cooldown
- reputation/renown
```
