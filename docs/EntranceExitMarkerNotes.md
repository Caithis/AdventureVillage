# Entrance and Exit Marker Notes

## v0.4.0 Implementation

The Town view now clearly marks:

```text
Town Entrance
World Exit
```

## Why This Matters

The player needs to understand where adventurers are coming from and where they leave for the World Map.

The current route is:

```text
Town Entrance
→ General Store / Inn
→ World Exit
```

## Future Direction

Later, entrance and exit markers should become part of the town layout system.

Possible future rules:
- Entrance is fixed by the kingdom road.
- World Exit connects to the frontier road.
- Roads can improve walking speed.
- Gates can be upgraded.
- Defensive structures may protect the World Exit.
