# Sidebar Viewport Layout Notes

## v0.5.5 Direction

The project window has been widened so the sidebar can sit outside the original gameplay viewport area.

## Current Layout

```text
0 - 1280 px:
    gameplay viewport / town area

1280 - 1600 px:
    sidebar lane
```

## Why This Matters

The game is becoming a management sim with many information layers.

Future sidebar modes may include:
- build menu
- building details
- debug tools
- economy trends
- adventurer list
- guild hall policies
- quest notices
- market/material trends

Keeping these in a sidebar prevents the town view from being buried under windows.
