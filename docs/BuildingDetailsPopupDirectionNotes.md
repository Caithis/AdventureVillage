# Building Details Popup Direction Notes

## Problem

The right sidebar Building Details panel is becoming cramped.

Guild Hall details in particular are too complex for a narrow sidebar.

## Recommended Direction

Building clicks should eventually open a larger popup in the main screen area.

The popup should:
- have a solid readable background
- have an X close button in the top-right
- support larger custom layouts
- allow per-adventurer action buttons beside each registry entry
- support aesthetic building-specific menus later

## Guild Hall Popup Needs

The Guild Hall popup should eventually contain:
- visitor intake controls
- regional cap info
- known adventurer registry
- per-adventurer favorite button
- per-adventurer priority return button
- per-adventurer contract button
- housing request/resident status

## Critical Scope Warning

Do not fully refactor all building UI until the current systems stabilize.

A popup refactor is worthwhile, but it should be its own patch.
