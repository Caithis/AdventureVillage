# Future Guild Registry and Renown Notes

## Visitor Pool Growth

The visitor pool should grow as the town's renown grows.

Possible renown sources:
- Guild Hall upgrades
- town level
- dungeon clears
- quest board reputation
- services offered
- resident adventurer achievements

## Why Pool Growth Matters

Early game should keep the pool manageable.

As the Guild Hall improves, the player should see new adventurer faces and classes.

## Guild Hall Registration

Adventurers should register at the Guild Hall when discovered.

This creates a known adventurer registry.

## Future Known Adventurer Actions

The player should eventually be able to:
- view known adventurers
- see visit history
- see satisfaction/happiness
- send a message/invitation using resources
- prioritize a known adventurer's return
- give gifts or support to encourage residency

## Priority Return

Priority return supports the favorite adventurer loop.

The player can invite a known adventurer back, help them improve, raise satisfaction, and eventually contract them as a resident.
