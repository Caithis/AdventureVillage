# Dungeon Frontier Guild-Town - GDD

## Current Version

v0.1.0 - Walking Skeleton

## Core Fantasy

Build and manage a frontier guild-town that supports adventurers, grows an economy around their needs, and survives escalating monster threats from the world map.

## Core Loop

Build town infrastructure → attract adventurers → sell supplies → adventurers explore/fight → adventurers return with loot → town buys materials → town crafts better goods → adventurers become stronger → stronger adventurers handle greater threats.

## Current Implemented Slice

v0.1.0 implements only the visible shell:
- Main scene
- Town scene
- World Map scene
- Debug UI
- GameClock
- GameState
- SceneRouter

## Next Design Target

The next design target is a placeholder adventurer who can spawn into town and move through a simple routine.
