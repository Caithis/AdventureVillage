# Naming Conventions

## Files and Folders

Use snake_case for resource files and lowercase folders.

Examples:
- `small_potion.tres`
- `slime_gel.tres`
- `grassland_edge.tres`

Use PascalCase for scene names and script class names when they represent major objects.

Examples:
- `Main.tscn`
- `WorldMap.tscn`
- `GameClock.gd`
- `Adventurer.gd`

## IDs

Use stable lowercase IDs for data.

Examples:
- `small_potion`
- `slime_gel`
- `general_store`
- `slime_nest`
- `grassland_edge`

## Git Branches

Recommended:
- `main`
- `dev`
- `feature/game-clock`
- `feature/debug-ui`
- `fix/adventurer-potion-purchase`

## Commit Messages

Recommended format:
- `docs: add project state`
- `feat: add game clock autoload`
- `fix: prevent negative inventory`
- `art: add placeholder adventurer sprite`
